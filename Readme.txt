A simple password strength tester built with HTML, CSS, and JavaScript.
This tool evaluates password strength in real time and provides feedback to help users
create stronger passwords.
Features
  Dynamically evaluates password strength based on:
  Length of the password
  Use of uppercase and lowercase letters
  Inclusion of numbers
  Use of special characters
  Provides visual feedback using a progress bar.
  Lightweight and responsive design.

Heres a link to checkout the page yourself:
https://jandreoli04.github.io/PasswordStrengthTester/

For more information on password strengths, check out this article from the CISA
https://www.cisa.gov/secure-our-world/require-strong-passwords
